Visual Identity
9.1 Logos

Sayada.ai Primary Logo

Stylized crown emblem incorporating an “S” shape, representing intelligence, leadership, and elevated decision-making.

Gradient blue treatment (from deep tech blue to lighter electric blue).

Paired with the wordmark “Sayada.ai” in a modern sans-serif (e.g., Montserrat Bold).

Standard lockups:

Crown above wordmark.

Crown to the left of wordmark for horizontal applications.

Usage:

Use SVG versions for all modern interfaces.

Prefer gradient logo on light or dark neutral backgrounds.

For small spaces (favicons, app icons, tool badges), use icon-only crown.

Parent Brand: Velocity Venture Holdings

Visual style: Futuristic, circuit-inspired technology motif with blue-silver accents.

Use language:
“Sayada.ai, a Velocity Venture Holdings, LLC company.”

Application:

Present as supporting parent brand in footers, about pages, pitch decks, legal notices, and onboarding flows.

When space allows, pair the Sayada.ai logo with the Velocity Venture Holdings logo in a compact “powered by / parent company” treatment.

Logo Protection Rules

Minimum height: 40 px for primary logo in digital contexts.

Clear space: Maintain at least the height of the letter “S” around all logo edges.

Never:

Distort, stretch, or rotate the logo.

Change colors outside the approved palette.

Place on busy or low-contrast backgrounds without a solid backing shape.

Always:

Use vector (SVG) assets where possible.

Request or generate single-color versions (white / dark) for emboss, engraving, or monochrome printing if needed.

9.2 Color Palette

Primary

Tech Blue — #2A2A72

Uses: headlines, primary CTAs, key accents, logo base.

Secondary

Silver Gray — #C0C0C0

Uses: borders, secondary text, UI chrome.

Accent

Gradient Blue — from #2A2A72 to #5A5AF2

Uses: logos, buttons, progress bars, data highlights.

Neutrals

Dark Background — #1A1A1A

White — #FFFFFF

Guidance

Use dark gradients (e.g., #1A1A1A → #2A2A72) for depth in hero sections, dashboards, and cards.

Maintain strong contrast for accessibility (WCAG 2.1 AA or better).

9.3 Typography

Headlines

Font: Montserrat (or comparable geometric sans-serif), Bold or SemiBold.

Suggested sizes (web):

H1: 48 px

H2: 36 px

H3: 28 px

Letter-spacing: around 0.02 em for a modern tech feel.

Body Text

Font: Inter (or comparable clean sans-serif).

Base size: 16 px.

Line-height: ~1.5 for readability.

Code / Technical Snippets

Font: Monaco or similar monospace.

Rules

Use no more than three font weights on a single page.

Maintain consistent type hierarchy across all Sayada.ai-powered products.

9.4 Imagery & Iconography

Style

Abstract, high-tech representations: circuits, data streams, connected nodes, stylized crowns, and motion lines.

Avoid generic stock photography whenever possible.

Prefer AI-generated or custom vector imagery that can be reused across assets.

Icons

Line-based or minimal solid icons.

Use blue gradients for key icons; neutrals for supporting icons.

Keep stroke weights consistent across a given surface.

9.5 UI / UX Guidelines

Layout

Modular, grid-based designs for easy scaling and reuse.

Mobile-first, responsive implementations across all viewports.

Interactions

Subtle micro-animations (e.g., button hover, confirmation ticks, loading indicators).

Use tooltips and inline help for complex concepts.

Aim for one-screen onboarding with clear role selection (e.g., founder, consultant, enterprise leader).

Accessibility

Meet or exceed WCAG 2.1 AA:

Color contrast ratio ≥ 4.5:1 for body text.

Fully keyboard-navigable controls.

Clear focus states and error messaging.

11.2 Messaging Rules

Prioritize precision; reference metrics and outcomes where possible.

Use active voice and keep most sentences under ~20 words.

Focus on business benefits (e.g., “Save 3 hours per week” vs. “Our tool is fast.”).

Avoid hype-driven language (e.g., “revolutionary,” “magic,” “unbelievable”).

Tie claims back to core principles: precision, speed, clarity, strength, scalability.

Example line:

“Analyze competitor pricing in seconds with precision data—unlock your next strategic edge.”
